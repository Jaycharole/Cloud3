Syllabus

  • Module 1: Introduction to the Course

    A general introduction to the course covering all the important aspects that you should know when you are in the Data Analytics industry. This module will be a basic module that will help you in framing your analytical perspective.

    • What is Data Analytics?
    • Need for Data Analytics in today’s Industry
    • Introduction to Problem Solving
    • Framing the problem
    • Analyzing the problem
    • Implementing the problem with a proper solution
    • Different problem-solving frameworks to adopt
    • Case study discussion

  • Module 2: Python for Data Analytics

    Python is one of the most important language when it comes to analytics. Due to rich python libraries data analysis tasks become quite easy and interesting. So in this module we are going to start with Python for Data Analytics where all the important needful concepts will be covered in detail.

    Basics of Python



    • Basics of Python
    • Data Structures in Python
    • Introduction to Problem Solving
    • Control Structures (if, if-Else, elseif, nested if-else, etc.)
    • Loops in Python
    • Functions
    • Object-Oriented Programming

    Python for Data Science



    • NumPy for Data Analytics
    • Pandas for Data Analytics
    • Introduction to Visualization
    • Matplotlib for Data Analytics
    • Seaborn for Data Analytics
    • BONUS: Plotly and Bokeh for Data Analytics

    Data Collection Methods



    • Introduction to Web Scrapping
    • Web Scrapping using various libraries
    • Some standards of Data Collection

    Exploratory Data Analysis (EDA)



    • Data Sourcing (Connecting various types of files)
    • Data Cleaning Techniques (Handling null values, outlier treatment, Filtering data, etc.)
    • Univariate Analysis
    • Bivariate Analysis
    • Multivariate Analysis


    MODULE PROJECT

    In the Module Projects we are going to perform two projects which are as follows:

    1) Web Scrapping Project
    In this project we are going to scrap data from different websites and store the data in our system in a standard format.

    2) Exploratory Data Analysis Project
    The detailed project where a popular datasets will be taken and detailed EDA will be performed. You will learn to list down various insights using this EDA project.

  • Module 3: Statistics for Data Analytics

    Data Analytics is all about statistical analysis and learning. In this module, we are going to learn about different statistical concepts which are not important for Data Analytics and also from an interview point of view.

    Introduction to Statistics



    • Type of statistics
    • Type of Data
    • Different Sampling Techniques
    • Measures of Central Tendency (Mean, Median, Mode)
    • Measures of Dispersion (Variance and Standard Deviation)

    Intermediate Statistics



    • Introduction to Probability
    • Permutations and Combinations
    • Conditional Probability and Bayes Theorem
    • Introduction to Gaussian Distribution and properties
    • Central Limit Theorem
    • Covariance and Correlation
    • Pearson And Spearman Rank Correlation
    • Binomial Distribution
    • Bernoulli Distribution
    • Poisson Distribution

    Advanced Statistics



    • Introduction to Confidence Intervals
    • Z-test and t-test
    • Hypothesis Testing
    • ANNOVA
    • AB Testing

    NOTE

    All the concepts of the module will be covered theoretically and practically as well.

  • Module 4: Microsoft Excel

    Data Analytics has evolved over years. But one of the basic analytical and data collecting tool even used today is Microsoft Excel. In this module we are going to learn about MS Excel in detail covering the important concepts.



    Introduction to MS Excel



    • Introduction to Excel Interface
    • Sorting and Filtering of Data
    • Formatting in Excel
    • Shortcuts in Excel

    Intermediate Excel



    • Complex Functions
    • Text functions
    • Formulas in Excel
    • Statistical functions in Excel
    • Charts in Excel
    • Pivot Table
    • VLOOKUP in Excel
    • Conditional Formatting

    Advanced Excel



    • LOOKUP functions
    • WHAT-IF Analysis
    • Dashboard in Excel
    • Basics of Macros
    • Advanced Visualization

    MODULE PROJECT


    Excel Dashboard Project
    In this module project, you will learn to develop a professional dashboard using Excel and whatever you have learned throughout the module.

  • Module 5: SQL and NoSQL

    Data is stored in Databases and databases can be of various types. But when we talk about Databases what first comes in our mind is SQL (Structured Query Language). Even in most of the data analytics job description SQL is always mentioned. So this is a crucial module focusing on SQL and basics of NoSQL.



    SQL



    • Introduction to Databases
    • What is SQL?
    • Introduction to Schemas and its types
    • Introduction to Relational and Non-Relational Schemas
    • Different integrity constraints in SQL
    • Details on different keys in SQL
    • Data Definition Language (DDL)
    • Data Manipulation Language (DML)
    • Data Control Language (DCL)
    • Joins in SQL
    • Data import and export
    • Functions in SQL
    • Nested Queries
    • Views in SQL
    • Stored Procedures
    • Window Functions
    • Python connectivity for SQL

    NoSQL



    • What is NoSQL databases?
    • Need of NoSQL databases
    • Introduction to MongoDB
    • Structure of MongoDB
    • Basics of Querying in MongoDB

    MODULE PROJECT


    Analyzing Data using SQL
    Here in this project you will be given a complex dataset where you will perform analysis using SQL.

  • Module 6: Power BI

    Power BI is a Microsoft product that is widely used for reporting and dashboard making. In this, we are going to cover Power BI which will help you to make professional dashboards for your stakeholders.



    • Introduction to Power BI and architecture
    • Connecting various Data Sources in PBI
    • Power Query Editor
    • M Language
    • Charts and Visuals in Power BI
    • Relationships in Power BI
    • DAX in Power BI
    • Custom and Advanced Visualization
    • Key Performance Indicators
    • Tool tips and Drill-down
    • Introduction to Power BI Service
    • Administration in Power BI
    • Security in Power BI
    • Data refresh in Power BI
    • Best Practices for making dashboards
    • EDA with Power BI

    MODULE PROJECT


    Power BI Dashboards
    In the project module you will learn to create 2 detailed dashboard which will help you to showcase your skills during interview.

  • Module 7: Tableau

    Power Similar to Power BI, Tableau is also one of the famous visualization and dashboard tool used in industry. Tableau has its own features which we will learn in this module.



    • Introduction to Tableau and its architecture
    • Intro to Tableau interface
    • Relationships, Joins and Unions
    • Sets in Tableau
    • Group in Tableau
    • Hierarchies in Tableau
    • Filters in Tableau
    • Highlighting
    • Device Designer in Tableau
    • Data Blending and MarkSize
    • Transparency
    • Data Aggregation
    • Generated fields
    • Charts in Tableau
    • Pivot Table in Tableau
    • LOD in Tableau
    • Calculated fields in Tableau
    • Formatting in Tableau
    • Forecasting in Tableau
    • Analytics in Tableau

    MODULE PROJECT


    Tableau Dashboards
    In LIVE lectures detailed Tableau dashboard will be created using the content you learned throughout the module.

  • Module 8: Predective Modelling

    Predictive analytics is one type of analytics. In this module, we will learn about the basics of Machine Learning where you will get the idea of how various ML algorithms work.



    • Introduction to Predictive Modelling
    • Introduction to Machine Learning
    • Supervised and Unsupervised Learning
    • Linear Regression
    • Regression metrics
    • Logistic Regression
    • Classification Metrics
    • K-Nearest Neighbors
    • Decision Trees
    • Dimensionality Reduction
    • Feature Engineering
    • Feature Selection

    MODULE PROJECT


    ML Project
    In this project you will implement ML algorithms that we have learned on a complex datasets along with EDA and pre-processing.

  • Module 9: Data Warehousing and ETL

    Here we provide you with a bonus track for you where you will learn about Data Warehousing and ETL.



    • Introduction to Data Warehouse

    • ETL vs ELT
    • Example using a Case study
